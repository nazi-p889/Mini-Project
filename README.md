# Emotion Detection using DeepFace

This project demonstrates **real-time and image-based emotion detection** using the **DeepFace** library and **OpenCV**.  
It detects the **dominant facial emotion** from an uploaded image (Google Colab).

---

## Features
- Detects facial emotions such as **happy, sad, neutral, angry, surprise**, etc.
- Supports:
  - **Image-based emotion detection** (Google Colab)
  - Automatically downloads and caches required DeepFace models.

---

## Technologies Used
- Python  
- DeepFace  
- OpenCV  
- Google Colab (for image-based testing)
